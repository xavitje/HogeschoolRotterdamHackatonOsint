# HogeschoolRotterdamHackatonOsint
# 🕵️ CMI OSINT Hackathon - Challenge #1: ShadowHunter

Welkom bij de eerste officiële uitdaging van de CMI OSINT Hackathon. In deze repository vind je de benodigde bestanden om de digitale voetafdruk van "ShadowHunter" te achterhalen.

## 🚀 Hoe begin je?

1. Ga naar de live website van deze challenge: `https://jouwgebruikersnaam.github.io/jouw-repo-naam/`
2. Lees de instructies in de terminal goed door.
3. Gebruik je OSINT-skills en de aanbevolen tools om de 5 vlaggen te vinden.

## ⚠️ Belangrijk: De Afbeelding (`wijnhaven107.png`)

De afbeelding in deze repository is niet zomaar een plaatje. Het bevat meerdere verborgen lagen, metadata en een versleutelde container.

**LET OP:** Je kunt de afbeelding **niet** correct analyseren door hem simpelweg in de browser van GitHub te bekijken. GitHub laat vaak een geoptimaliseerde preview zien waarbij verborgen data verloren gaat.

### Hoe download je het originele bestand?
Om de puzzels op te lossen, **moet je de afbeelding downloaden** naar je eigen computer:
1. Klik op het bestand `wijnhaven107.png` in de lijst hierboven.
2. Klik op de knop **"Download"** (of de drie puntjes -> "Download") om het originele, onbewerkte bestand op te slaan.
3. Gebruik tools zoals Notepad, 7-Zip en steganografie-sites op het gedownloade bestand.

## 🛠️ Aanbevolen Tools
Onderaan de live website vind je een "Gereedschapskist" met links naar handige tools zoals CyberChef, StegOnline en DNS-lookup diensten.
